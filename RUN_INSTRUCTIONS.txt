┌───────────────────────────────────────────────────────────────┐
│ OPTION 2: Manual Setup in VS Code (RECOMMENDED)              │
└───────────────────────────────────────────────────────────────┘

Step 1: Open in VS Code
  - Open VS Code
  - File → Open Folder → Select this folder

Step 2: Open Terminal in VS Code (Ctrl+`)

Step 3: Create Virtual Environment
  python -m venv venv

Step 4: Activate Virtual Environment
  # Mac/Linux:
  source venv/bin/activate

  # Windows:
  venv\Scripts\activate

Step 5: Install Dependencies
  pip install -r requirements.txt

Step 6: Get OpenAI API Key
  - Visit: https://platform.openai.com/api-keys
  - Sign up/login
  - Create new secret key
  - Copy it (starts with sk-...)

Step 7: Configure Environment
  - Copy .env.example to .env
  - Edit .env file
  - Add: OPENAI_API_KEY=sk-your-key-here
  - Save file

Step 8: Run Tests
  python main.py test

Step 9: Run Dashboard (Optional)
  python app.py
  # Visit: http://localhost:5000

═══════════════════════════════════════════════════════════════════
 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

"Module not found" error:
  → Make sure virtual environment is activated
  → Re-run: pip install -r requirements.txt

"OPENAI_API_KEY not found":
  → Check .env file exists (not .env.example)
  → Verify key is correctly formatted: OPENAI_API_KEY=sk-...
  → No quotes around the key

"Port 5000 already in use":
  → Kill existing process or use different port
  → Edit app.py, change: app.run(port=5001)

APIs not responding:
  → Check internet connection
  → Some APIs have rate limits (normal on first try)
  → Wait 1-2 seconds and try again
